Hexed tiles

Hexed is, so far, mostly a board game idea, but one that we believe strongly in. The goal of the game is to cast spells by building specific shapes using energy represented by hexagonal tiles. We created the icons for each energy type with recognizability and the ability to tell the energy types apart as the focus.
